# 基数排序（桶排序）
## 算法思路
1.找出最长的数字 确定要处理的桶排序的趟数（位数）
2.依次由个位开始处理 把相应位数上的数字放入相应序号的桶里面 完成后 再按照桶的序号 依次取出桶里面的数据 放回原始的数组当中
```cpp
mod = 10;
dev = 1;
for(int i = 0;i < len ;++i, mod*= 10, dev*= 10){
    int index = (number % mod)/dev;
}
```
## 桶的数据类型
- vector<vector<int>>
## 缺陷
1. 没有办法统一一段代码同时实现整形和浮点型的数据排序
2. 不好处理负数（可以用20个桶去放 将下标加上10即可）

## 性能参数
1. 时间复杂度: O(nd) (d是最长数字的位数)
2. 空间复杂度：O(n)
3. 稳定性：稳定
